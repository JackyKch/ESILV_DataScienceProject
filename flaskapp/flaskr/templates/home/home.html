Data Science Project


Hi ! Welcome on the API for ou data science project. This API let you build and compare different types of classifier. On this home page you can also find our results on finding the best classifier. We tried multiple comparison. First we compared the four classifier with the default values for the hyperparameters. We found the following results on the test dataset:

Classifier Accuracy
BaggingClassifier 0.886
RandomForestClassifier 0.893
DecisionTreeClassifier 0.8485
LogisticRegression 0.8395

We then tried to compare the BaggingClassifier and the RandomForestClassifier

Classifier Accuracy
BaggingClassifier(base_estimator=DecisionTreeClassifier(), n_estimators=15) 0.8855
RandomForestClassifier(n_estimators=15) 0.8945

As we can see the RandomForestClassifier is slightly more performant than the BaggingClassifier. We tested different number of estimators for those two classifier and found that 100 was the one giving the best performances.
